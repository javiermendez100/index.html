TAREAS ADMINISTRATIVAS Y SEGURIDAD

CAMBIO DE CONTRASEÑA

Cualquier usuario puede cambiar su contraseña usando el comando "passwd", de la siguiente manera:

passwd

Tenga en cuenta que el comando no requiere argumentos.

El administrador puede cambiar la contraseña de cualquier usuario. Por ejemplo, suponga que el administrador tiene que cambiar la contraseña de un usuario cuyo login es lisa. Esto se puede lograr escribiendo lo siguiente comando:

passwd lisa

Una cosa interesante es que el sistema operativo obliga a los usuarios habituales a crear contraseñas largas, con un mínimo de 5 o 6 caracteres; mientras que el administrador puede crear contraseñas de cualquier longitud. Por ejemplo, si un usuario normal intenta crear la contraseña "123", el sistema no lo permitirá; pero el administrador podría hacerlo.

CREAR Y BORRAR USUARIOS

Solo los administradores pueden crear y eliminar cuentas.

Para crear una cuenta, use el comando "adduser". Por ejemplo, para crear una cuenta cuyo nombre de usuario (login) es lisa, usa el comando:

adduser lisa

Este comando crea automáticamente el directorio de inicio y el grupo del usuario, ambos llamados lisa. Además, durante la ejecución del comando, el sistema operativo solicitará la introducción del contraseña que se utilizará para ingresar a la cuenta, así como otra información menos relevante como: nombre completo del usuario, número de teléfono, etc.

Por otro lado, para eliminar una cuenta se utiliza el comando “deluser”. Por ejemplo, para eliminar un cuenta cuyo nombre de usuario (login) es lisa, use el comando:
deluser lisa

Con este comando se elimina el grupo principal de la cuenta, es decir, el grupo lisa. Pero, el directorio de inicio ubicado en el directorio de inicio permanece. Para eliminar el directorio de inicio debemos agregar la opción “--remove-home”, de la siguiente manera:

deluser --remove-home lisa
En cualquier caso, el administrador siempre puede eliminar el directorio de inicio de la cuenta mediante el comando:

rm-r lisa

CREAR Y BORRAR GRUPOS

Solo los administradores pueden crear y eliminar grupos.

Para crear un grupo, utilice el comando "groupadd". Por ejemplo, para crear un grupo llamado estudiantes vamos a usar el siguiente comando:

groupadd estudiantes
Por otro lado, para eliminar grupos se utiliza el comando “groupdel”. Por ejemplo, para eliminar al grupo de estudiantes usaremos el siguiente comando:
gruoupdel estudiantes

AÑADIR O ELIMINAR USUARIOS DEL GRUPO

Solo los administradores pueden agregar y eliminar usuarios de los grupos. Para hacer que un usuario pertenezca a varios grupos, además del propio, se utiliza el comando "usermod" con las opciones a y G. Por ejemplo, para que la usuaria lisa pertenezca a los grupos de profesores y tutores, la Se usarán los siguientes comandos:

usermod -aG profesores lisa usermod -aG tutores lisa

Estos dos comandos se pueden reducir a uno separando los nombres de los grupos con comas:

usermod -aG profesores, tutores lisa

Tenga en cuenta que no hay espacios en blanco entre los nombres de los grupos y la coma de separación. Por otro lado, para eliminar un usuario de un grupo el comando "deluser". Por ejemplo, para quitar el usuario cuyo inicio de sesión es lisa del grupo de tutores, use el siguiente comando:

deluser tutores lisa

CAMBIAR EL GRUPO DE UN ARCHIVO O DIRECTORIO

Por defecto, un archivo o directorio pertenece al grupo del usuario que lo crea. Para cambiar al grupo de el archivo, utilice el comando "chgrp". Por ejemplo, para que el archivo test.txt pertenezca al grupo de profesores, usa el comando:

prueba de maestros chgrp.txt

Para cambiar el grupo de un directorio y todo su contenido se utiliza la opción R. Por ejemplo:

prueba de maestros chgrp -R

hace que el directorio de prueba y todo su contenido pertenezcan al grupo de profesores.
